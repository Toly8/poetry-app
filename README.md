# poetry-app

Многостраничное PWA-приложение для iPhone и других устройств:

- `index.html` — добавление стихотворений;
- `collection.html` — коллекция с поиском, редактированием и удалением;
- офлайн-хранение и очередь синхронизации (`IndexedDB + outbox`).

## Возможности

- Добавление стихов офлайн.
- Просмотр своей коллекции.
- Поиск по названию и тексту.
- Редактирование и удаление стихотворений.
- Автоматическая синхронизация операций (`add`, `update`, `delete`) при появлении интернета.

## Запуск

```bash
python3 -m http.server 8080
```

Откройте:
- `http://localhost:8080/index.html`
- `http://localhost:8080/collection.html`

## Важно

В `app.js` используется `API_BASE_URL = "https://example.com/api/poems"`.
Замените этот адрес на ваш реальный backend API.
