# poetry-app

Многостраничное PWA-приложение для iPhone и других устройств:

- `index.html` — добавление стихотворений;
- `collection.html` — коллекция с превью и полноэкранным чтением;
- `reader.html` — отдельная вкладка с настройками режима чтения;
- офлайн-хранение и очередь синхронизации (`IndexedDB + outbox`).

## Возможности

- Добавление стихов офлайн.
- Поля: название, автор, текст стихотворения.
- В коллекции показывается короткое превью: название, автор и несколько строк (предпросмотр). Если текст длиннее — добавляется многоточие, а полная версия открывается только по кнопке «Читать полностью».
- Полное чтение открывается по нажатию на карточку в полноэкранном режиме.
- Выход из полноэкранного чтения:
  - свайп слева направо;
  - либо пролистать вверх и нажать кнопку «Свернуть».
- При прокрутке вниз в полноэкранном чтении шапка (название, автор, кнопка) скрывается.
- Поиск по названию, автору и тексту.
- Редактирование и удаление стихотворений.
- Автоматическая синхронизация операций (`add`, `update`, `delete`) при появлении интернета.
- Отдельная вкладка «Чтение» с понятными анимированными переключателями:
  - комфортный режим чтения;
  - режим только чтения (без кнопок);
  - тёмная тема.
- Настройка шрифта: увеличение/уменьшение и выбор семейства.
- Автор отображается под названием стихотворения небольшим серым шрифтом.

## Запуск

```bash
python3 -m http.server 8080
```

Откройте:
- `http://localhost:8080/index.html`
- `http://localhost:8080/collection.html`
- `http://localhost:8080/reader.html`

## Важно

В `app.js` используется `API_BASE_URL = "https://example.com/api/poems"`.
Замените этот адрес на ваш реальный backend API.
